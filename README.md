# CNC-fault-diagnosis-using-BN
CNC machine fault diagnosis using BN and arduino

This work is the framwork to the research conducted during my phd thesis, witch consists of CNC machine fault diagnosis system 
using Bayesian networks formalism.
Please feel free to use the code source and files. If you want to use this work in your research, please cite it. You can consult
the link below to find citing informations.
